# SoftwareAction UI 美化实施方案（v2 详细版）

> 蘑菇大棚 IoT 管理系统界面升级 — 双主题 + 毛玻璃 + 动效
> 基于 HarmonyOS 官方设计规范 + 现代设计语言
> 更新日期：2026-06-28

---

## 一、设计目标

1. **双主题切换**：雾透玻璃（年轻活力）+ 极致黑白（专业克制）
2. **毛玻璃质感**：Header / 卡片 / TabBar 三处统一使用 `backdropBlur + 半透 + 白边`
3. **完整动效系统**：页面进入 / 卡片出现 / 按钮按下 / 数值过渡 / 状态脉冲
4. **零硬编码颜色**：所有颜色从 `ThemeManager.getTokens()` 读取，原子替换
5. **性能可控**：低端设备自动降级毛玻璃，限制同屏 backdropBlur 元素数量

#### 毛玻璃核心公式

> **blur vs backdropBlur（重要）**
> - `.backdropBlur(n)` — 模糊**下层内容**，毛玻璃效果用这个
> - `.blur(n)` — 模糊**组件自身内容**，装饰光晕用这个
> - 毛玻璃场景严禁用 `.blur()`

```typescript
// 雾透卡片 — 三件套：半透白底 + backdropBlur 模糊下层 + 白边
Column() { /* 内容 */ }
  .borderRadius(16)
  .backgroundColor('rgba(255, 255, 255, 0.65)')
  .backdropBlur(16)
  .border({ width: 1, color: 'rgba(180, 200, 220, 0.35)' })
  .shadow({ radius: 12, color: '#7EC8A325', offsetY: 4 })
```

#### 动效规范

| 场景 | 实现 | 参数 |
|------|------|------|
| 页面进入 | `animateTo` opacity 0→1 + translateY 20→0 | 400ms springMotion(0.4, 0.8) |
| 卡片出现 | 逐个 stagger 弹出 | 每张间隔 60ms，scale 0.9→1.05→1.0 |
| 按钮按下 | 弹性反馈 | springMotion scale 1→0.92→1.0，200ms |
| 数值变化 | 数字滚动过渡 | animateTo 800ms EaseOut |
| 页面切换 | 淡入 + 上滑 | 300ms EaseInOut |
| 传感器状态 | 光晕脉冲（正常）/ 呼吸闪烁（告警） | 2s 循环 opacity 0.3↔0.6 |

### 3.2 传感器卡片 — 毛玻璃 + 状态光晕

```typescript
@Builder
SensorCard(item: SensorDataItem) {
  Stack() {
    // 状态光晕（根据 status 变色）— 装饰光晕用 .blur() 模糊自身
    Ellipse()
      .width('80%')
      .height(30)
      .fill(this.getStatusGlowColor(item.status))
      .blur(20)
      .opacity(0.4)
      .position({ x: '10%', y: '70%' })

    Column() {
      // 图标 + 状态点
      Row() {
        Text(SensorConfigUtil.getConfig(item.type).icon).fontSize(22)
        Blank()
        Circle({ width: 8, height: 8 })
          .fill(this.getStatusColor(item.status))
          .shadow({ radius: 4, color: this.getStatusColor(item.status), offsetY: 0 })
      }
      .width('100%')

      // 数值 + 单位（数值用断点自适应字号，不用 token 固定字号）
      Row() {
        Text(item.value.toFixed(1))
          .fontSize(AdaptiveUtils.getSensorDataFontSize(this.currentBreakpoint))
          .fontColor(this.t.textTitle)
          .fontWeight(FontWeight.Bold)
        Text(SensorConfigUtil.getConfig(item.type).unit)
          .fontSize(this.t.fontSm)
          .fontColor(this.t.textSub)
          .margin({ left: 4, bottom: 6 })
      }
      .alignItems(VerticalAlign.Bottom)
      .margin({ top: 8 })

      Text(SensorConfigUtil.getConfig(item.type).name)
        .fontSize(this.t.fontSm)
        .fontColor(this.t.textSub)
        .margin({ top: 4 })
    }
    .width('100%')
    .padding(16)
  }
  .width('100%')
  .borderRadius(this.t.radiusMd)
  .backgroundColor(this.t.bgCard)        // 半透白底（雾透）或纯白（黑白）
  .backdropBlur(this.t.blurAmount)       // 毛玻璃模糊下层（黑白主题=0）
  .border({ width: 1, color: this.t.border })
  .shadow({
    radius: this.t.shadowCardRadius,
    color: this.t.shadowCardColor,
    offsetY: this.t.shadowCardOffsetY
  })
}

// 状态光晕颜色
private getStatusGlowColor(status: number): string {
  switch (status) {
    case 0: return 'rgba(67, 160, 71, 0.5)';   // 绿色光晕
    case 1: return 'rgba(255, 143, 0, 0.5)';    // 琥珀光晕
    case 2: return 'rgba(229, 57, 53, 0.5)';    // 红色光晕
    default: return 'rgba(67, 160, 71, 0.3)';
  }
}
```

### 3.6 控制按钮 — 脉冲动画

```typescript
@State pulseScale: number = 1;

@Builder
ControlButton(item: ControlItem) {
  Column() {
    Stack() {
      Text(item.icon).fontSize(28)
      if (item.active) {
        Circle({ width: 10, height: 10 })
          .fill(this.t.success)
          .position({ x: 28, y: 2 })
      }
    }
    .width(52)
    .height(52)
    .borderRadius(26)
    .backgroundColor(item.active ? item.color + '20' : this.t.bgCardLight)
    .border({ width: item.active ? 2 : 0, color: item.color })
    .scale({ x: this.pulseScale, y: this.pulseScale })

    Text(item.name)
      .fontSize(this.t.fontSm)
      .fontColor(item.active ? item.color : this.t.textSub)
      .margin({ top: 6 })
  }
  .onClick(() => {
    // 触发脉冲动画
    animateTo({
      duration: 300,
      curve: Curve.EaseOut,
      onFinish: () => { this.pulseScale = 1; }
    }, () => {
      this.pulseScale = 1.3;
    });
    this.onControlToggle(item);
  })
}
```

## 四、ThemeManager 实现

### 4.1 ThemeTokens 接口

```typescript
// features/core/src/main/ets/theme/ThemeTokens.ets

/**
 * 完整设计令牌接口
 *
 * 设计原则：每新增一个颜色或数值，必须同时在两个主题中定义，
 * 切换时原子替换，不会出现混合状态。
 *
 * 字号说明：本接口中的 fontXs~fontXxl 用于「小字号场景」
 * （标题/正文/辅助文字/单位/标签等）。传感器数值、概览数字、图表数据等

  readonly colorLight: string;

  readonly colorPh: string;
  // 圆角
  readonly radiusSm: number;
  readonly radiusMd: number;
  readonly radiusLg: number;
  readonly radiusXl: number;
  readonly radiusFull: number;
  // 阴影
  readonly shadowCardRadius: number;
```

### 4.4 ThemeManager 单例

```typescript
// features/core/src/main/ets/theme/ThemeManager.ets
import { ThemeTokens } from './ThemeTokens';
import { FROSTED_THEME } from './ThemeFrosted';
import { CONTRAST_THEME } from './ThemeContrast';
import Preferences from '@ohos.data.preferences';
import { common } from '@kit.AbilityKit';

const PREFS_KEY = 'current_theme';
const PREFS_NAME = 'app_settings';

/**
 * 主题管理器单例
 *
 * 职责：
 * 1. 持有当前主题令牌
 * 2. 从本地存储恢复主题
 * 3. 切换主题并通知全局
 *
 * 用法：
 *   const t = ThemeManager.getInstance().getTokens();
 *   Text('hello').fontColor(t.textTitle)
 */
export class ThemeManager {
  private static instance: ThemeManager;
  private current: ThemeTokens = FROSTED_THEME;
  private prefs: Preferences | null = null;

  private constructor() {}

  static getInstance(): ThemeManager {
    if (!ThemeManager.instance) {
      ThemeManager.instance = new ThemeManager();
    }
    return ThemeManager.instance;
  }

  /**
   * 初始化：从本地存储恢复主题
   *
   * 必须在 EntryAbility.onCreate 中调用，传入 ability context。
   */
  async init(context: common.Context): Promise<void> {
    this.prefs = await Preferences.getPreferences(context, PREFS_NAME);
    const saved = await this.prefs.get(PREFS_KEY, 'frosted');
    this.current = (saved === 'contrast') ? CONTRAST_THEME : FROSTED_THEME;
    AppStorage.setOrCreate<string>('themeName', this.current.name);
  }

  /** 获取当前完整令牌 */
  getTokens(): ThemeTokens {
    return this.current;
  }

  /** 当前是否雾透主题 */
  isFrosted(): boolean {
    return this.current.name === 'frosted';
  }

  /** 当前是否黑白主题 */
  isContrast(): boolean {
    return this.current.name === 'contrast';
  }

  /**
   * 切换主题 — 原子替换
   *
   * 切换后写入本地存储 + 通知 AppStorage，所有 @StorageLink('themeName')
   * 的组件会自动触发 build() 重绘。
   */
  async toggle(): Promise<void> {
    const next = this.isFrosted() ? CONTRAST_THEME : FROSTED_THEME;
    this.current = next;
    if (this.prefs) {
      await this.prefs.put(PREFS_KEY, next.name);
      await this.prefs.flush();
    }
    AppStorage.setOrCreate<string>('themeName', next.name);
  }

  /**
   * 直接设置主题
   *
   * @param themeName 'frosted' 或 'contrast'
   */
  async setTheme(themeName: string): Promise<void> {
    const next = (themeName === 'contrast') ? CONTRAST_THEME : FROSTED_THEME;
    this.current = next;
    if (this.prefs) {
      await this.prefs.put(PREFS_KEY, next.name);
      await this.prefs.flush();
    }
    AppStorage.setOrCreate<string>('themeName', next.name);
  }
}
```

### 5.1 毛玻璃降级

低端设备 `backdropBlur` 会卡顿，检测策略：

```typescript
// 在 ThemeManager 中增加设备能力检测
private shouldUseBackdropBlur(): boolean {
  // 方案 1：通过 deviceInfo 检测
  // 方案 2：通过首帧渲染时间判断
  // 方案 3：用户在设置中手动关闭
  return this.current.blurAmount > 0;
}

// 降级后的卡片背景（不 backdropBlur，但保留半透）
getCardBackground(): string {
  if (this.shouldUseBackdropBlur()) {
    return this.current.bgCard;  // 'rgba(255,255,255,0.65)'
  }
  return '#FFFFFF';  // 降级为纯白
}
```

### 5.2 同屏 backdropBlur 限制

| 设备档次 | 同屏 backdropBlur 元素上限 |
|---------|------------------|
| 高端（8GB+） | ≤8 个 |
| 中端（4-8GB） | ≤5 个 |
| 低端（<4GB） | ≤2 个 或 关闭 |

DashboardPage 实际 backdropBlur 元素：
- Header 背景：1
- 概览卡片：1
- 传感器卡片：最多 6
- 底部 TabBar：1
- **合计 9 个** → 低端设备需降级

> 注：Header 内的装饰光晕用 `.blur()`（模糊自身），不属于 backdropBlur 计数。

## 十、关键约束（不可违反）

1. **零硬编码颜色**：组件中不允许出现 `'#XXXXXX'` 或 `'#XXXXXX' + '20'`，必须从 `this.t` 读取
2. **Token 完整性**：每新增一个颜色/数值，必须同时在两个主题中定义
3. **原子替换**：主题切换时整组 Token 替换，不允许部分替换
4. **性能优先**：低端设备 backdropBlur 自动降级，动画可关闭
5. **无 any 类型**：ThemeTokens 接口字段必须明确类型
6. **动画只动复合属性**：`scale / translate / opacity`，禁止动画 `width / height / margin`
7. **毛玻璃用 `backdropBlur`，光晕用 `blur`**：
   - 卡片/Header/TabBar 的毛玻璃效果 → `.backdropBlur(this.t.blurAmount)`（模糊下层内容）
   - 装饰光晕（Ellipse 模糊自身边缘）→ `.blur(n)`（模糊自身）
   - 严禁混用，严禁毛玻璃场景用 `blur`
8. **字号分两级管理**：
   - **小字号场景**（标题/正文/辅助文字/单位/标签）→ 用 `this.t.fontXs/fontSm/fontMd/fontLg/fontXl`
   - **大字号场景**（传感器数值/概览数字/图表数据）→ 用 `AdaptiveUtils.getXxxFontSize(breakpoint)` 断点自适应
   - 严禁大字号场景用 token 固定字号（手机端会过大）
